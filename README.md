# Calculator

> Well, I made this simple calculator, made in C++, it has functions to add, divide, multiply, and subtract.

## How to compile

Make sure you have CMake installed on your PC.

```bash
# Set up the build directory
cmake -B build -DCMAKE_BUILD_TYPE=Release

# Compile the project
cmake --build build

```

## How to execute
```bash
./build/Release/simple-calculator
```
